## How To Run
1. Make sure you have already install docker
1. Make sure DB Port `:3306` not conflict with your local machine
1. Or you can change in `docker-compose.yml` file
1. Run docker compose 
```
  docker-compose up -d
```
1. Stop docker compose with
```
  docker-compose down
```
